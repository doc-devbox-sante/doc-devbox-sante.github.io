Interpreter les logs

Gestion des traces dans la DevBox-santé DMP

Obtenir les logs

Les logs sont générés dans un fichier output.log dans le cas d’un installer ou directement dans la console pour une instance docker.

Pour s’assurer que les traces SOAP soient bien générés dans les traces, il faut que la configuration des logs de org.apache.cxf.services :

  • soit à un niveau info dans les versions jusqu’à 5.0
  • soit à un niveau debug dans les versions postérieures (à partir de la 5.1)

Pour forcer un niveau de log : /6.x/connecteurs/dmp/configuration/#logging
