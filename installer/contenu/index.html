Configuration

La DevBox-santé est paramétré par défaut lors de l’installation, mais elle est personnalisable en modifiant le fichier application.yml. Les modifications possibles de cette configuration est décrite pour chacun des composants dans leur espace de documentation :

  • pour le DMP : ../../dmp/configuration
  • pour l’INSi : ../../insi/configuration
