Différents environnements du DMP

Les différents environnements pour accéder au DMP

Pour le DMP, Ils existent quatre types d’environnements

Attention

Retour sur le webinaire du 10 octobre à 14h00 - Présentation des évolutions du GI version 2.9.1 :

Les différents environnements dédiés aux éditeurs vont être supprimés, en contrepartie en phase d’homologation chaque éditeur aura son propre jeu de patients sur l’environnement DEV1.

Deprecated

Cette page est le fruit de l’interpération des fiches d’informations : 

  • PDT-INF-547 - Paramétrage des URL v5 (29/04/2024)
  • PDT-INF-506v6 - Mise à niveau des environnements de formation (25/01/2024)

Nous essayons autant que se peut maintenir à jour ces informations. En cas de doute, vous pouvez trouver la dernière version de ces informations sur votre espace industriel : https://industriels.sesam-vitale.fr

Environnements de Production

  • LPS v2 : https://lps2.dmp.gouv.fr/si-dmp-server/v2/services
  • Web PS via CPS : https://wps-cps.dmp.monespacesante.fr
  • Web PS via PSC : https://wps-psc.dmp.monespacesante.fr

Environnement de Développement

  • Accessible à tous les utilisateurs l’url LPS : https://lps.dev1.dmp.gouv.fr/si-dmp-server/v2/services
  • Environnement Web PS accès CPS : https://wps-cps.dev1.dmp.gouv.fr/ps/acces-web
  • Environnement Web PS accès PSC : https://wps-psc.dev1.dmp.gouv.fr/ps/acces-web

Environnements de Formation

Dans la fiche d’information PDT-INF-506v6 en date du 10 janvier 2024, il est fait mention de 4 environnements de formation avec des rétentions de données différentes :

Formation 1 - Jeu de données jamais réinitialisé

  • LPS : https://lps.formation1.dmp.gouv.fr/si-dmp-server/v2/services
  • WebPS via CPS : https://wps-cps.formation1.dmp.gouv.fr/ps/acces-web
  • WebPS via PSC: https://wps-psc.formation1.dmp.gouv.fr/ps/acces-web
  • Web Patient : https://formation1.patient.dmp.gouv.fr

Formation 3 - Jeu de données réinitialisé tous les jours

  • LPS : https://lps.formation3.dmp.gouv.fr/si-dmp-server/v2/services
  • WebPS via CPS : https://wps-cps.formation3.dmp.gouv.fr/ps/acces-web
  • WebPS via PSC: https://wps-psc.formation3.dmp.gouv.fr/ps/acces-web
  • Web Patient : https://formation3.patient.dmp.gouv.fr

Formation 4 - Jeu de données réinitialisé tous les 15 jours

  • LPS : https://lps.formation4.dmp.gouv.fr/si-dmp-server/v2/services
  • WebPS via CPS : https://wps-cps.formation4.dmp.gouv.fr/ps/acces-web
  • WebPS via PSC: https://wps-psc.formation4.dmp.gouv.fr/ps/acces-web
  • Web Patient : https://formation4.patient.dmp.gouv.fr

Formation 5 - Jeu de données réinitialisé tous les mois

  • LPS : https://lps.formation5.dmp.gouv.fr/si-dmp-server/v2/services
  • WebPS via CPS : https://wps-cps.formation5.dmp.gouv.fr/ps/acces-web
  • WebPS via PSC: https://wps-psc.formation5.dmp.gouv.fr/ps/acces-web
  • Web Patient : https://formation5.patient.dmp.gouv.fr

Environnement de Développement

  • Accessible à tous les utilisateurs l’url LPS : https://lps.dev1.dmp.gouv.fr/si-dmp-server/v2/services
